# 📖 About AttendX

AttendX is a smart attendance management application built specifically for college students.

Instead of marking attendance manually every lecture, students simply enter:

- 📚 Their Subjects
- 📅 Weekly Timetable
- 🎓 Semester Start Date
- 🎓 Semester End Date

AttendX automatically predicts and tracks attendance throughout the semester.

If a class is missed or an extra class is attended, users can quickly update it from the Calendar section.

The objective of AttendX is to remove the repetitive work of attendance calculation while helping students maintain the required attendance percentage.

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/RajputPraful/attendx.git
```

Move into the project

```bash
cd attendx
```

---

## Backend Setup

```bash
cd backend

pip install -r requirements.txt

uvicorn app.main:app --reload
```

Backend runs on

```
http://127.0.0.1:8000
```

---

## Flutter Setup

```bash
cd frontend

flutter pub get

flutter run
```
